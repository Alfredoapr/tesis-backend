Buscar tesis por query
Descripcion

Devuelve una lista con las tesis encontradas que coincidan con los parametros enviados

URL
        GET /thesis/search

        query {
            tags=
            name=
            year=
            thesisId=
            school=
            status=
            resume=
        }
Ejemplo
GET /thesis/search?tags=ia,redes&year=2021
