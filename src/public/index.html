Tesis

Descripcion de Tesis

Modelo

const thesisSchema = new Schema({
    thesisId: String,
    name: String,
    year: Number,
    tags: [String],
    school: String,
    resume: String,
    pdfRoute: String,
    status: Number,
    statusMessage: String,
    synodals: [String],
    director: String,
    authors: [String]
});

Rutas

Lista de tesis
Descripcion

Devuelve la lista completa de todas las tesis existentes

URL
GET /thesis
Crear una tesis
Descripcion

Crea una nueva tesis con los datos enviados y la devuelve

URL
POST /thesis
{
    thesisId: String,
    name: String,
    year: Number,
    tags: [String],
    school: String,
    resume: String,
    pdfRoute: String,
    status: Number,
    statusMessage: String,
    synodals: [String],
    director: String,
    authors: [String]
}
Buscar tesis por query
Descripcion

Devuelve una lista con las tesis encontradas que coincidan con los parametros enviados

URL
        GET /thesis/search

        query {
            tags=
            name=
            year=
            thesisId=
            school=
            status=
            resume=
        }
Ejemplo
GET /thesis/search?tags=ia,redes&year=2021
